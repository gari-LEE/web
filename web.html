온라인 바로예약 (실시간 예약) FAQ

📋 온라인 바로예약이란?
고객이 직접 원하는 일정과 시간으로 예약함에 따라 확정되는 실시간 예약시스템입니다.

❓ 어떻게 바로예약 할 수 있나요? +
📍 예약 경로
① 자생한방병원 홈페이지 통한 예약
② 카카오톡 자생한방병원 채널 / 예약알림톡 통한 예약

🖥️ 예약 방법

  • 1. 홈페이지 - PC: 홈페이지 우측 하단에서 예약
  • 2. 홈페이지 - 모바일: 모바일 홈페이지에서 예약
  • 3. 카카오톡: 자생한방병원 채널/알림톡 통한 예약
PC 홈페이지 우측 하단 '바로예약' 버튼
모바일 홈페이지 '바로예약' 버튼
카카오톡 알림톡 예약 화면
💡 참고사항
• ①, ② 방법 모두 가능
• 예약변경 희망 시에는 ② 방법(카카오톡)을 통해서만 가능합니다.
⚙️ 어떤 절차로 예약이 되나요? +
🆕 초진 예약 절차
희망 병의원 선택 -> 이름/ 주민번호(앞,뒤 13자리 입력) 입력-> 연락처 등록 -> 보험종류(건보, TA, 의료급여, 일반) 선택 -> 질환선택 → 의료진 선택(예진의 선택가능) → 희망일자 선택 → 희망시간 선택 → 예약확정
🔄 재진 예약 절차
재진 진료내역 팝업 화면에서 선택:
  • ① 건강보험 → 교통사고 변경 : 희망의료진 선택 → 희망일자 선택 → 희망시간 선택 → 재초진 예약 확정
  • ② 다른지역 예약 : 희망병의원 선택 → 이름/주민번호 재입력 → 연락처 등록 → 보험종류 선택 →
    • 의료진 선택 → 희망일자 선택 → 희망시간 선택 → 확정
  • ③ 의료진 변경 : 의료진 변경 사유 선택 → 질환선택 → 의료진 선택 → 희망일자 및 희망시간 선택 → 확정
💡 참고사항
• ① 기존 담당의료진이 있을 시 담당원장이라고 확인되며, 다른 의료진 선택도 가능합니다.
• ③ 의료진 변경 선택 후 예약 시 "[TS]의료진 변경사유"가 환자메모에 자동 기재됩니다.
🔄 오프라인 예약과 온라인 예약을 같이 할 수 있나요? +

📋 예약 조합 가능 케이스

  • ① 예약이 없는 경우
    온라인예약을 통해 예약 가능
  • ② 오프라인 1건 예약 있는 상황
    온라인예약으로 변경 가능
  • ③ 오프라인 1건 예약 되어 있는 상태
    온라인 예약 1개 추가 예약 가능
  • ④ 오프라인 2건 예약 중
    1건의 예약을 온라인 예약을 통해 변경 가능
  • ⑤ 오프라인 1건 + 온라인 예약 1건
    온라인예약을 다시 변경 가능
💡 정리
오프라인 예약(상담예약팀)과 온라인 예약(바로예약)은 유연하게 조합하여 이용하실 수 있습니다.
🤱 처음 가는 임산부는 바로예약 할 수 없나요? +
⚠️ 바로예약 불가 대상
다음의 경우 바로예약을 통한 확정예약이 불가합니다:
  • 임산부 (첫 진료시)
  • 외국인
  • 산업재해
  • 골절 관련 첫 진료
  • 양방 진료 희망
  • 검사를 희망하는 경우
📞 참고 사항
전화번호 남기기를 통해 연락처 기재 후 상담예약팀 통한 예약으로 진행해 주세요.

[담당 주치의 미설정] -> 전화번호 남기기 통해 연락처 기재 후 [상담예약팀 통해 확정 처리 필요]

  • ▶ 담당주치의 미설정인 경우
    바로예약 불가
    예약 진행시,
    [담당의료진이 확인되지 않습니다]
    팝업 이후 상담 접수화면으로 이동
  • ▷ 담당주치의 설정된 경우
    바로예약 가능
※ SWS - 온라인 바로예약 전화신청통해 접수건 확인 후 아웃콜 진행
👨‍⚕️ 예약하려는데 담당원장님 시간이 비활성화 되어 있어요 +
🚫 온라인 바로예약 불가 의료진
다음의 경우 온라인 바로예약을 통한 예약이 불가합니다:
  • 퇴사예정 의료진
  • 대표원장님
💡 확인 방법
PM MEMO 상 "온라인예약 OFF"로 메모 확인 가능
📞 예약 방법
대표번호를 통한 오프라인 예약만 가능합니다.
⏰ 예약하려는데 원하는 시간을 선택할 수 없어요 +

🔍 시간 선택 불가 사유

① 해당시간 예약 마감
이미 예약이 가득 찬 시간대입니다.
② 1시간 이내 예약 제한
예약시점 시간 기준 1시간 이후 시간만 예약 가능
(1시간 이내 시간은 선택 불가)
③ 병의원별 시간 제한
병의원에 따라 12시/17시 예약시간은 온라인 바로예약이 어려울 수 있습니다.

-> 특정 의료진 혹은 예진의 클릭 후 의사예약일정 통해 예약 가능 허용수 확인 가능

💡 해결 방법
[사진첨부 설명]
온/오프 재진 (초/재초진)
▷ 온라인, 오프라인 예약이 가능한 허용수(예약된 수 X)

오프 재진 (초/재초진)
▷ 오프라인(콜센터)만 예약 가능한 허용수(예약된 수 X)

(오전) 해당 의료진의 경우,
온/오프라인 예약 가능시간대 : 9시, 9시반, 10시, 10시반, 11시 예약
오프라인 예약 가능 시간대 : 11시30분, 12시 예약

예시
  • 9시 재진의 경우 - 온라인, 오프라인 예약 중 해당시간 선착순 3명까지 허용
    3명 허용수에서 현재 재진예약 3건으로 예약 마감
    (예약마감으로 온라인 비활성화)
  • 10시 재진의 경우 - 온라인, 오프라인 3명의 허용수 중 재진예약 2건 진행됨
    온라인 예약 혹은 오프라인으로 1명 추가 진행 가능(온라인 활성화)
  • 11시 30분 재진의 경우 - 온라인 예약 불가
    오프라인 예약을 통해 현재 오프라인 예약 1명 되어있어
    오프라인 예약 2명 추가예약 가능(온라인 비활성화)
